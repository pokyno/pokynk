# Pokynk

A portable, Inkscape-compatible **SVG viewer/editor** that runs entirely in your
browser — the engine is a single WebAssembly component, the UI is a thin static
shell. No server, no account; your files stay on your device.

### ▶ Live: https://pokyno.github.io/pokynk/
